# Исправление электронного дневника

Данный скрипт поможет исправить твои плохие оценки в электронном дневнике, удалить замечания и получить похвалы от учителей.

## Как установить

- Поместить файл `scripts.py` в ту же папку, где расположена база данных электронного дневника `schoolbase.sqlite3`
- Если хочется протестировать скрипт на своём ПК понадобится файл с базой данных и развёрнутый сайт:
    * Скачать [сайт](https://github.com/devmanorg/e-diary/tree/master)
    * Скачать или создать свою собственную базу данных
    * Запустить сайт на своем ПК, следуя инструкциям из [README сайта](https://github.com/devmanorg/e-diary/tree/master)

## Как подготовиться к использованию функций скрипта

- Команды для навигации по папкам через консоль:

| Действие | Windows (CMD) | Windows (PowerShell) | Linux/macOS |
|----------|---------------|---------------------|-------------|
| Список файлов | `dir` | `Get-ChildItem` или `ls` | `ls` |
| Войти в папку | `cd папка` | `Set-Location папка` или `cd папка` | `cd папка` |
| На уровень выше | `cd ..` | `Set-Location ..` или `cd ..` | `cd ..` |
| Очистить экран | `cls` | `Clear-Host` или `cls` | `clear` или `Ctrl+L` |

- Открой командную строку и перейди в папку, содержайщую файлы `scripts.py` и `schoolbase.sqlite3`
- Используй следующую команду, чтобы запустить интерактивную консоль:

```powershell

```

```powershell

```

## Функции и команды для их запуска:
- `get_schoolkid` используется внутри других функций для поиска ученика в Базе Данных по "Фамилии Имени Отчеству"

```powershell
scripts.get_schoolkid(kid_name)
```

- `fix_marks` исправит все двойки и тройки на пятерки
    * Вместо `kid_name` впишите "Фамилию Имя Отчество", либо просто "Фамилию Имя"

```powershell
scripts.fix_marks(kid_name)
```

- `remove_chastisements` удалить все замечания учителей из твоего дневника
    * Вместо `kid_name` впишите "Фамилию Имя Отчество", либо просто "Фамилию Имя"

```powershell
scripts.remove_chastisements(kid_name)
```

- `create_commendation` создаст похвалу в твоем дневнике
    * Вместо `kid_name` впишите "Фамилию Имя Отчество", либо просто "Фамилию Имя"
    * Вместо `subject` впишите название предмета, по которому нужно получить похвалу
    * Не забывай про заглавные буквы и не удаляй кавычки

```powershell
scripts.create_commendation(kid_name, subject)
```
